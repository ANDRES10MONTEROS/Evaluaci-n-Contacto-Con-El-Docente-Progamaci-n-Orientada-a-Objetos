Manual de Instalación y Ejecución

1. Requisitos
- Go 1.20+
- Git

2. Clonar repositorio
git clone https://github.com/TU_USUARIO/streaming-go.git
cd streaming-go

3. Instalar dependencias
go mod tidy

4. Ejecutar servidor
go run ./cmd/app

Servidor disponible en:
http://localhost:8080/api/


Todo el uso se encuentra documentado en endpoints.md
